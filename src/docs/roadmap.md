# Roadmap

## Vision à 3 ans (2024 — 2026)

### 📅 Phase 1 — Fondations (2024)

- [x] Définition du concept et de l'architecture
- [x] Pitch deck et analyse concurrentielle
- [x] Prototypage du Notitia Node (ESP32-S3)
- [x] Mise en place de l'infrastructure d'équipe (Linear, Discord, GitHub)
- [ ] POC transcription Whisper sur mobile
- [ ] Premier prototype de l'application mobile

### 📅 Phase 2 — MVP (2025 H1)

- [ ] Application mobile fonctionnelle (iOS + Android)
- [ ] Transcription en temps réel
- [ ] Résumé automatique basique
- [ ] Synchronisation Bluetooth avec le Node
- [ ] Système de consentement (LED + QR Code)
- [ ] Chiffrement E2EE de bout en bout

### 📅 Phase 3 — Meduza (2025 H2)

- [ ] Assistant IA conversationnel
- [ ] Recherche sémantique dans l'historique
- [ ] Génération de Mindmaps
- [ ] Intégration Notion
- [ ] Beta publique

### 📅 Phase 4 — Production (2026 H1)

- [ ] Lancement officiel de l'application
- [ ] Commercialisation du Notitia Node
- [ ] Plans Gratuit et Premium
- [ ] Intégrations Slack + Calendriers
- [ ] Support Apple Watch
- [ ] Analytics personnels avancés

### 📅 Phase 5 — Expansion (2026 H2)

- [ ] API publique pour développeurs
- [ ] Marketplace d'intégrations
- [ ] Support multi-plateforme (Desktop)
- [ ] Modèles IA améliorés
- [ ] Programme entreprise (B2B)

## Modèle Économique

### Plan Gratuit
- Fonctionnalités de base
- Données supprimées après **7 jours**
- Mémoire éphémère

### Plan Premium — 9,99€/mois
- Archive **illimitée**
- Mindmaps avancées
- Recherche sémantique complète
- Fonctions collaboratives
- Support prioritaire

### Notitia Node — ~149€
- Hardware dédié
- Souvent inclus : période Premium gratuite
