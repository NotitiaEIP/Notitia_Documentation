# 🌐 Infrastructure & Benchmark Cloud

### **Notitia — Année 1 (2026)**

</div>

---

Ce document détaille les besoins en ressources cloud et distribution pour la première année de l'EIP **Notitia**.  
Notre stratégie repose sur le **Edge AI**, minimisant les coûts serveurs au profit de la confidentialité.

<br>

## 📊 Benchmark : Edge AI vs Cloud Classique

L'innovation de Notitia est de **décentraliser le calcul**. Voici pourquoi notre infrastructure est plus rentable que celle de la concurrence *(Otter.ai, Plaud Note, etc.)* :

<br>

| Critère | ☁️ Architecture Cloud | ⚡ Architecture Notitia |
|:--------|:----------------------|:-----------------------|
| **Coût Serveur IA** | ~3 600 $/mois *(1000 users)* | **0 $** — Calcul sur smartphone |
| **Confidentialité** | Données sur serveurs tiers | **Zero-Knowledge** *(100% local)* |
| **Dépendance Réseau** | Requiert internet pour l'IA | **Offline First** |
| **Rendu Final** | Texte / Résumé | **Mindmap Interactive** |

<br>

## 🛠️ Services Cloud & Infrastructure
