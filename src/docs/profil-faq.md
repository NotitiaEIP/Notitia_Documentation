# Profil, Abonnements & FAQ

## Meduza — Ta mascotte IA

**Meduza** est une méduse animée qui te donne des retours visuels en temps réel.

| État | Quand | Visuel |
|---|---|---|
| 👋 Bonjour | Splash screen / Onboarding | Animation douce |
| 🎙️ Écoute | Pendant l'enregistrement | Cyan pulsant |
| ⚙️ Traitement | Correction IA / Génération | Rotation + progression |
| 😊 Succès | Note sauvegardée | Vert, rebond |
| 😕 Erreur | Problème de connexion | Rouge, tremblement |

Meduza n'est visible **que lors des événements importants** — elle n'est pas affichée en permanence.

## Profil et Paramètres

Accède au profil via le **bouton avatar** en haut à droite.

### Ce que tu peux modifier

- **Pseudo** affiché dans l'app
- **Photo de profil** (depuis la galerie ou l'appareil photo)
- **Langue** : Français / English
- **Thème visuel** : 5 thèmes disponibles
  - 🟣 Cyberpunk (rose néon + bleu nuit) — défaut
  - 🔵 Ocean (cyan + bleu marine)
  - 🟢 Forest (vert néon + vert sombre)
  - 🟠 Sunset (orange + marron)
  - 🟣 Violet (violet + indigo)
- **Déconnexion** du compte

## FAQ

**Q : Mes notes sont-elles sauvegardées si je n'ai pas de compte ?**
R : Oui, en mode invité les notes sont sauvegardées **localement**. Elles ne sont pas synchronisées et peuvent être perdues si tu réinstalles l'app.

**Q : Est-ce que Notitia fonctionne hors ligne ?**
R : Partiellement. La transcription Deepgram nécessite internet. Le moteur natif fonctionne hors ligne. La correction IA et l'assistant nécessitent internet.

**Q : Comment accéder au widget depuis mon écran d'accueil ?**
R : Sur Android : appui long → Widgets → Notitia. Sur iOS : vue Aujourd'hui → ajouter le widget.

**Q : La réunion fonctionne-t-elle à distance ?**
R : Non, la réunion utilise le réseau local (Wi-Fi). Tous les participants doivent être sur le même réseau.

**Q : Mon NFC ne fonctionne pas ?**
R : Vérifie que le NFC est activé dans Paramètres > Connexions > NFC.
