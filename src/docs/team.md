# L'Équipe

## Organisation

L'équipe Notitia est composée de **6 membres** étudiants d'Epitech, répartis selon les domaines d'expertise.

## Répartition

| Domaine | Membres | Focus |
|---------|---------|-------|
| 🔧 **Hardware** | 3 membres | ESP32-S3, firmware, conception PCB |
| 🤖 **IA / Software** | 2 membres | Whisper, LLM, application mobile |
| 🔒 **Sécurité / UX** | 1 membre | Chiffrement, design, expérience utilisateur |

## Outils & Workflow

### Gestion de Projet
- **Linear** — Cycles (Sprints) et tickets de développement
- **GitHub** — Dépôts privés avec webhooks Discord

### Communication
- **Discord** — Serveur structuré avec channels dédiés
- **DailyBot** — Check-ins hebdomadaires (Lundi)
- **Sesh** — Gestion des deadlines Epitech

### Développement
- **GitHub Actions** — CI/CD automatisé
- **Webhooks** — Suivi en temps réel des commits et Pull Requests
- **Code Reviews** — Revue obligatoire avant merge

## Valeurs d'Équipe

1. **Transparence** — Communication ouverte et honnête
2. **Qualité** — Code propre, testé et documenté
3. **Innovation** — Repousser les limites du possible
4. **Éthique** — La technologie au service de l'humain
